# Folder Images

Simpan semua gambar produk di folder ini.

## Cara Menambahkan Gambar:

1. **Copy gambar** ke folder `public/images/`
2. **Nama file:** gunakan nama yang jelas, contoh: `sepatu-running-nike-1.jpg`
3. **Format:** JPG, PNG, atau WebP
4. **Ukuran:** Disarankan maksimal 1MB per gambar

## Cara Menggunakan di Form Produk:

Setelah gambar disimpan di `public/images/`, gunakan URL:
- `/images/nama-file.jpg`

Contoh:
- File: `public/images/sepatu-nike.jpg`
- URL: `/images/sepatu-nike.jpg`

## Tips:

- Gunakan nama file yang deskriptif
- Hindari spasi, gunakan tanda hubung (-)
- Contoh: `sepatu-running-nike-air-zoom.jpg`
